# ReparaTech - Sistema de Gestión Full Stack ##

## Descripción corta del sistema ####
Una plataforma web integral diseñada para centralizar la gestión de solicitudes técnicas y el seguimiento de reparaciones de hardware en tiempo real.

## Roles
* **Técnico Líder:** Administra las órdenes de trabajo y actualiza los estados del servicio.
* **Cliente:** Consulta el progreso de su equipo de manera externa.

## MVP ###
1. Registro de ingresos y diagnósticos.
2. Dashboard de control de estados.
3. Buscador de tickets para clientes.
