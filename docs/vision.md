# Visión del Proyecto: Sistema de Gestión ReparaTech

## Problema
La falta de un sistema digitalizado para el seguimiento de reparaciones técnicas genera demoras en la comunicación con el cliente y desorden en el inventario de repuestos.

## Usuarios y roles
*   **Técnico Líder:** Gestiona las órdenes de trabajo y actualiza los estados de reparación (ej. "En diagnóstico", "Listo para entrega").
*   **Cliente:** Consulta el avance de su equipo de forma remota.

## MVP (Mínimo Producto Viable)
*   Módulo de ingreso de equipos con registro de fallas.
*   Panel de estados para técnicos.
*   Buscador básico para que el cliente vea el estado de su ticket.
